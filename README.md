# Redux Thunk Infinite Posts

A simple React app that fetches paginated posts from JSONPlaceholder using Redux Toolkit `createAsyncThunk`.

## Features

- Redux Toolkit store with async thunk
- Paginated REST API calls with Axios
- Load-more post list UI
- Basic loading and error states

## Tech Stack

- React + Vite
- Redux Toolkit
- React Redux
- Axios

## Run Locally

```bash
npm install
npm run dev
```

Then open `http://localhost:5173`.

## Build

```bash
npm run build
```
